# Coding Challenges

A collection of solutions to various LeetCode and CodeWars problems, along with my personal study notes.

## Approach

- For non-trivial problems, I include alternative solutions alongside deep-dive analysis and complexity explanations. I like reminding myself of what's possible.

- Some folders contain companion `.md` files where I sketched out ideas and conceptual overviews during the process.

- To keep my skills sharp across multiple stacks, I use a script (`randlang.py`) that randomly rolls a language for each new LeetCode problem. Take a look at the script to see the current rotation.
